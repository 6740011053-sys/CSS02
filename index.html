รายวิชา: 934-206 Web Programming

การตกแต่งปุ่มด้วย CSS

งานชิ้นนี้เป็นการฝึกใช้ CSS เพื่อปรับแต่งปุ่มให้ดูสวยงาม และสามารถโต้ตอบกับผู้ใช้ได้เมื่อมีการเลื่อนเมาส์

  • กำหนดสีและขนาดของปุ่ม
  • เพิ่มระยะห่างภายในปุ่ม
  • ทำให้ขอบปุ่มโค้งมน
  • เพิ่มเอฟเฟกต์เมื่อชี้เมาส์
